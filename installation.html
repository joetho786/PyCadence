Getting Started¶

Prerequisites¶

  • OS : Linux/CentOS or any other Unix based OS

  • Python 3.6 or later

  • Cadence Virtuoso with Ocean Scripting enabled

  • tmux installed

Installation¶

To install PyCadence, run the following command:

pip install CadenceConnector
